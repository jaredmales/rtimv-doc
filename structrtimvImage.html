Detailed Description

Base class for rtimv images.

Defines the interface to an image stream to be displayed by rtimv.

virtual void rtimvImage::detach ( )

Detach / disconnect from the image source and clean up.

Primarily used in the event of a SIGSEGV or SIGBUS. This instance must be ready to connect again, and begin monitoring the source for new data.
