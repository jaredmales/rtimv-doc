size_t rtimvDictBlob::getBlobStr ( char *  blob,
size_t  mxsz,
timespec *  ts = nullptr

inline

Get the data stored in the blob as a C string.

Copies the smaller of mxsz and m_sz bytes to blob.

This guarantees that blob[sz-1] = 0 where sz is the number of bytes copied. If the returned size == mxsz then you should not trust the contents unless you know that the correct size was copied or don't care if the string is truncated.

If m_blob has 0 size or is nullptr, then this sets blob[0] = 0 and returns 1.

Returns
the number of bytes copied
Parameters
blob[in/out] The pointer to the array to copy the data to
[in]mxszThe max number of bytes to copy to blob
[out]ts[optional] if not null this will be filled in with the time of last modification
